# A06

This folder contains all the files related to the A06 assignment.
The main script `graph.py` loads data from `graph.csv`, plots it and saves the plot into `graph.png`.

## Instructions to run the script

```cmd
:: clone this repository
git clone https://github.com/kalinapetrcuni/ci2.git

:: create a python virtual enviroment
python -m venv A06_env

:: activate the enviroment
A06_env\Scripts\activate

:: install dependencies
pip install -r requirements.txt

:: navigate into the directory A06\ within the repository
cd ci2\A06\

:: run the script
python graph.py
```
