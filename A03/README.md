# ci2 - task A03

This folder contains my solution of the A03 task on HTML/XML parsing in Python.

The script `pubchem_parser.py` extracts IUPAC names of compounds from PubChem search result XML files and prints them to stdout.

## Commands to run the script (on Windows in cmd)

```
python -m venv <name_of_your_enviroment> # create python virtual enviroment

# navigate into <name_of_your_enviroment>\Scripts

activate.bat # activate the enviroment

pip install -r requirements.txt # install the required packages

python pubchem_parser.py <name_of_target_file>.xml # run the script

```
